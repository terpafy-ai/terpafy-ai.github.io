## Font Families

| Font | Source | Usage |
|------|--------|-------|
| **Anton** | Google Fonts | Display / headings — all-caps, high-impact |
| **Inter** | Google Fonts | Body text, UI elements, secondary headings |
| **Gotham** | Licensed | Button labels (in design system components) |

### Font Import (Google Fonts)
```
https://fonts.googleapis.com/css2?family=Anton&family=Inter:wght@400;700&display=swap
```

> **Note**: Gotham is a licensed font used in the Figma button component specs. For web, consider using Inter (which is already in the design system) as a substitute for button text.
