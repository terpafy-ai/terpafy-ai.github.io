## Grid System

The design system defines a **12-column grid**:

| Property | Value |
|----------|-------|
| Total width | 1512px |
| Columns | 12 |
| Column width | 91px |
| Gutter | 20px |
| Margin (offset) | 100px each side |
| Alignment | STRETCH |
| Content width | 1512 − (2 × 100) = **1312px** |
